## Getting Started

### Installation

1.  Clone the repo

```sh
git clone https://github.com/GHOSCHT/gif-browser.git
```

2.  Install NPM packages

```sh
yarn
```

## Development server

Start the development server

```sh
yarn develop
```

## Packaging

Package for your desired platform

```sh
npm pack:{platform}
```

replace {platform} with one of the following platforms: win, mac, linux
