## Getting Started

### Installation

1.  Clone the repo

```sh
git clone https://github.com/GHOSCHT/gif-browser.git
```

2.  Install NPM packages

```sh
yarn
```

## Development server

Start the development server

```sh
yarn dev
```

## Packaging

Deploy to GitHub Pages

```sh
npm package
```
